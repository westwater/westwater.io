# Westwater Consulting

IT consulting business website built with React and Vite.

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS + shadcn/ui components
- Framer Motion (animations)
- React Router DOM

## Development

### Setup

```bash
npm install
```

### Dev Server

```bash
npm run dev
```

Opens at http://localhost:8080 (or next available port)

### Build

```bash
npm run build
```

Output goes to `dist/`

### Preview Production Build

```bash
npm run preview
```

## Deployment

Automatically deploys to GitHub Pages via `.github/workflows/deploy.yml` on push to master.

**Live site**: https://www.westwater.io
